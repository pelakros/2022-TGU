# 2022-Data-analyst-course-by-National-Research-Tomsk-State-University
Курс профессиональной переподготовки по программе «Data-аналитик: старт карьеры» национального исследовательского Томского государственного университета в период с 13.05.2022 по 14.09.2022 года в очно-заочной форме общим объёмом 256 часов.
